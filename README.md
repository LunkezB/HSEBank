## Описание программы

Даннное консольное приложение предназначено для учёта финансов, позволяет работать с счетами, категориями, операциями,
выдавать аналитику, экспортировать и импортировать Json и CSV.

Консольное меню:

1) Создать счёт
2) Переименовать счёт
3) Удалить счёт
4) Список счетов
5) Создать категорию
6) Удалить категорию
7) Список категорий
8) Создать операцию
9) Удалить операцию
10) Список операций
11) Аналитика: чистый доход за период
12) Аналитика: группировка по категориям
13) Аналитика: Топ-N категорий
14) Аналитика: динамика по месяцам
15) Экспорт в CSV
16) Экспорт в JSON
17) Импорт из CSV
18) Импорт из JSON
19) Undo (Отменить последнюю выполненную команду, если она это позволяет)
20) Redo (Восстановить отменённую с помощью Undo команду)
21) Повторить последнюю выполненную команду
22) Выход

## Запуск программы

```bash
git clone https://github.com/LunkezB/HSEBank
cd HSEBank/HSEBank
dotnet run
```

## Паттерны

- **Фасад**: AccountsFacade, AnalyticsFacade, CategoriesFacade, OperationsFacade.
- **Фабрика**: DomainFactory - создание счетов, категорий и операций.
- **Команда**: Все классы внутри Commands.
- **Посетитель**: IExportVisitor, CsvExporter, JsonExporter, IVisitable - для экспорта CSV и JSON в CSVImporter и JsonImporter соответственно.
- **Декоратор**: TimingDecorator, TimingUndoableDecorator - для подсчёта времени выполнения команд
- **Репозиторий**: BankAccountRepository, CategoryRepository, OperationRepository - для хранения данных
- **Внедрение зависимостей (Dependency Injection)**: использование ServiceCollection в Program. Используется NuGet-пакет Microsoft.Extensions.DependencyInjection.
